## [1.0.0] - 2025-01-12

### Added

- Initial release of home-mcp
- MCP server implementation for Apple Home on macOS
- Control HomeKit devices via the Model Context Protocol
- Scene management and activation
- Device status querying
- Integration with Claude and other AI assistants
- TypeScript support with full type definitions
- Comprehensive test suite with Vitest
- MIT license
